# 02. 用户使用指南与实战手册：打造你的职业蓝图

## 2.1 快速开始

### 界面概览
应用界面划分为三个核心区域：
1.  **顶部导航栏**：包含 AI 顾问开关、导出图片功能。
2.  **中央画布区**：经典的九宫格布局，支持响应式设计（移动端为标签页切换）。
3.  **右侧 AI 侧边栏**：与 DeepSeek 智能顾问对话的交互区域。

### 基础操作
- **添加卡片**：
    - 点击任意板块标题栏右侧的 `+` 按钮。
    - 或者双击板块内的空白区域。
    - 自动生成一张随机颜色的便利贴。
- **编辑内容**：
    - 点击便利贴内的文本即可开始编辑。
    - 支持自动保存，无需手动确认。
- **管理卡片**：
    - **拖拽**：按住便利贴可将其拖动到任意板块，重新组织思路。
    - **删除**：鼠标悬停在便利贴上，点击右上角的 `x` 删除。
    - **修改颜色**：(高级功能) 每次新建时随机分配颜色，用于区分不同维度的想法。

## 2.2 实战演练：绘制你的第一张画布

建议按照以下顺序填充画布，这符合逻辑推演过程：

### 第一阶段：盘点自我 (后台视角)
1.  **填写【核心资源】**：
    - *思考*：我最擅长什么？我有哪些独特的经历？
    - *操作*：添加卡片，如“精通 Vue.js”、“拥有PMP证书”、“性格外向善于交际”。
2.  **填写【关键业务】**：
    - *思考*：利用上述资源，我每天主要在忙什么？
    - *操作*：添加卡片，如“编写前端代码”、“组织每日站会”、“撰写技术文档”。

### 第二阶段：定义价值 (核心视角)
3.  **填写【客户群体】**：
    - *思考*：谁在消耗我的工作成果？
    - *操作*：添加卡片，如“产品经理”、“最终用户”、“技术总监”。
4.  **填写【价值主张】**：
    - *思考*：我为客户带来了什么好处？
    - *操作*：添加卡片，如“提升页面加载速度”、“减少沟通成本”、“确保项目按时交付”。

### 第三阶段：交付与互动 (前台视角)
5.  **填写【渠道通路】**：
    - *思考*：如何交付价值？
    - *操作*：添加卡片，如“Git提交”、“周报”、“技术分享会”。
6.  **填写【客户关系】**：
    - *思考*：如何维系信任？
    - *操作*：添加卡片，如“及时响应Bug”、“定期复盘”。

### 第四阶段：财务与成本 (底线视角)
7.  **填写【收入来源】** & **【成本结构】**：
    - *操作*：诚实记录你的薪资、奖金以及时间投入、压力源。

## 2.3 AI 智能顾问的使用技巧

本项目集成了 **DeepSeek-V3** 模型，它可以作为你的私人商业顾问。

### 配置 AI
1.  点击右上角 **设置图标 ⚙️**。
2.  输入你的 DeepSeek API Key（数据仅保存在本地）。
3.  点击保存。

### 高效提问指令 (Prompt)
- **诊断模式**：
    - *"分析我的画布，指出我的价值主张和客户群体是否匹配？"*
    - *"我的核心资源是技术，但想转型管理，请基于当前画布给出建议。"*
- **创意模式**：
    - *"基于我的核心资源，帮我构想3个可能的副业方向。"*
    - *"如何优化我的渠道通路，提升在团队中的影响力？"*
- **一键添加建议**：
    - AI 回复中生成的建议卡片，点击即可直接**飞入**左侧画布，无需手动复制粘贴。
    - 系统会自动去重，避免重复添加。

## 2.4 数据管理与隐私
- **自动保存**：所有修改实时保存至浏览器 `LocalStorage`。刷新页面不丢失。
- **数据重置**：在设置中点击“重置所有数据”，可一键清空画布、聊天记录和 API Key。
- **导出分享**：点击顶部“导出图片”按钮，生成高清单张图片，便于在简历、PPT或社交媒体中分享。

## 2.5 常见问题 (FAQ)
- **Q: 为什么刷新后 AI 建议的按钮变灰了？**
  - A: 系统会自动检测画布上是否已存在该建议。如果已存在，按钮置灰；如果您删除了画布上的卡片，按钮会自动恢复高亮，允许再次添加。
- **Q: 更换浏览器后数据还在吗？**
  - A: 不在。数据存储在当前浏览器的本地缓存中。如需迁移，请手动复制内容。
